DEFINICIÓN: PROYECTO DE SOFTWARE

Un **proyecto de software** abarca todo el procedimiento de desarrollo, desde la recogida de requisitos hasta las pruebas y el mantenimiento, llevado a cabo según metodologías específicas para lograr el producto deseado.

Tipos de Proyectos

APLICACIONES WEB

Software que se ejecuta directamente en el navegador web. Esencial para el intercambio de información y la provisión de servicios remotos de forma segura.

APLICACIONES MÓVILES

Software diseñado para ejecutarse en dispositivos móviles (smartphones o tabletas). Suelen ser unidades pequeñas con funciones específicas y de calidad.

APLICACIONES PARA ESCRITORIO

Programas o herramientas instaladas y utilizadas únicamente en ordenadores de sobremesa o portátiles, limitando su uso a ese dispositivo.

SISTEMAS EMBEBIDOS

Sistemas de computación diseñados para funciones específicas, con componentes integrados en una placa base como los que se encuentran en electrodomésticos o vehículos.

DESARROLLO DE VIDEOJUEGOS

Proceso creativo que va desde la idea inicial hasta la versión final del juego, usando herramientas específicas para materializar la experiencia de usuario.

CICLO DE VIDA DEL SOFTWARE Y SUS FASES

Proceso sistemático (SDLC) que guía una aplicación desde su concepción hasta su mantenimiento, asegurando que cumpla con los requisitos del usuario y sea funcional.
