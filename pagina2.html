APLICACIONES WEB: Frontend y Backend

Una aplicación web se compone de dos partes esenciales: el **Frontend** (la interfaz de usuario) y el **Backend** (la lógica del servidor y datos).

FRONTEND

Funciones Principales:

  • 1 **Presenta la Información:** Muestra los datos organizados y atractivos.
  • 2 **Permite la Interacción:** Gestiona clics, formularios y navegación.
  • 3 **Responde a Acciones:** Actualiza la interfaz en tiempo real para una experiencia dinámica.

Tecnologías Clave:

  • HTML Define la estructura (encabezados, párrafos).
  • CSS Estiliza la página (colores, fuentes, diseño).
  • JavaScript Agrega interactividad (animaciones, efectos).

BACKEND

Funciones Principales:

  • 1 **Procesamiento de Datos:** Recibe solicitudes, procesa y realiza operaciones necesarias.
  • 2 **Gestión de Bases de Datos:** Almacena, recupera y actualiza los datos persistentes.
  • 3 **Lógica de Negocio:** Implementa las reglas que rigen el funcionamiento interno de la aplicación.

Tecnologías y Frameworks:

  • Python (Django, Flask)
  • JavaScript (Node.js, Express.js)
  • Java (Spring Boot)
  • Ruby (Ruby on Rails)
  • PHP (Laravel, Symfony)
