PROYECTO DE SOFTWARE

Un framework es un conjunto de reglas y convenciones que se usan para desarrollar software de manera más eficiente y rápida. Estos marcos de trabajo se emplean para ahorrar tiempo y esfuerzo en el desarrollo de aplicaciones, ya que proporcionan una estructura básica que se puede utilizar como punto de partida.

Ventajas y Desventajas

VENTAJAS

  • **Agilidad y rapidez en el desarrollo.** Precisamente gracias a la reutilización de código mencionada anteriormente, conseguimos mayor rapidez en el desarrollo, ya que no perderemos tiempo en desarrollar funcionalidades nuevas.
  • **Minimizar errores y mayor facilidad para solucionarlos.** Como el framework ya incorpora código implementado por otros programadores, los posibles errores que este pueda tener siempre serán menores que al desarrollarlo desde cero. Además, en caso de que hubiera un error, lo más probable es que ya haya sido solucionado por la comunidad.

DESVENTAJAS

  • **Tiempo de aprendizaje.** Antes de empezar a utilizar un framework debemos familiarizarnos con él, con cómo se estructuran sus archivos, con la forma en la que se comunican los componentes, etc. Por lo tanto, tendremos que invertir tiempo en superar la curva de aprendizaje para poder comenzar un nuevo desarrollo utilizando el framework.
  • **Versiones inestables.** El hecho de que los frameworks sean tan populares provoca que estén en constante actualización para cumplir con las nuevas tecnologías y las nuevas políticas de seguridad. Por ello, si en el desarrollo surgen incompatibilidades con otras librerías o se detectan errores de seguridad, la elección de una versión muy reciente del framework podría ralentizarnos.

Frameworks Más Usados

FRAMEWORK FRONTEND MAS USADOS

  • **React:** Desarrollado por Facebook, React es uno de los más populares. Se destaca por su enfoque en componentes reutilizables, lo que facilita la construcción de interfaces complejas. Además, su gran comunidad y ecosistema de herramientas lo respaldan.

FRAMEWORK BACKEND MAS USADOS

  • **Django (Python):** Uno de los más populares y robustos, Django ofrece una estructura completa para desarrollar aplicaciones web de forma rápida y segura. Es ideal para proyectos de gran escala y complejos.
  • **Laravel (PHP):** Muy utilizado en la comunidad PHP, Laravel destaca por su elegante sintaxis y su enfoque en la convención sobre configuración. Es una excelente opción para proyectos de tamaño mediano y grande.
  • **Express.js (JavaScript):** Construido sobre Node.js, Express.js es un framework minimalista y flexible que te permite crear APIs y aplicaciones web de forma rápida. Es ideal para proyectos pequeños y medianos, así como para microservicios.
